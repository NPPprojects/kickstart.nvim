# Manual Apply Commands

`<Space>;` reject the current hunk

`u` undo the most recent manual-apply action, then fall back to normal undo

`<Space>b` autocomplete the current line during an insert phase

`<Space>\` approve the current line within the current hunk

`<leader>i` materialize the next guided insert line

`<leader>.` jump to the next hunk

`<leader>,` jump to the previous hunk

`<leader>m` clear the current manual-apply overlay
